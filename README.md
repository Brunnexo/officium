# Officium

O Officium nasceu da necessidade de automatizar processos administrativos dentro da indústria automobilística. O projeto está aberto para se adaptar à outras necessidades.

## Instalação

Necessário ter o [Node.JS](https://nodejs.org/en/) e [Microsoft SQL Server](https://www.microsoft.com/pt-br/sql-server/sql-server-downloads) instalados

Após o download e instalação, executar:
```cmd
npm install
```

## Uso

```cmd
cd <projeto>
electron .
```
